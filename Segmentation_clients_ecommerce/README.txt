README :

Liste des fichiers dans le projet:

Données:

Online Retail.xlsx - Dataset original
dataset.csv - Même dataset, mais au format .csv (utilisé dans le projet)
train.csv - Jeu d'entraînement utilisé pour la classification (sauvegardé à la fin du notebook Exploration)
test.csv - Jeu de test pour la classification (sauvegardé à la fin du notebook Exploration)
customers_full.csv - Jeu de données entier utilisé à la fin du projet pour entraîner le modèle sur l'ensemble des données
new_clients.csv - Jeu de données de test pour tester le programme de classification 'classify.py'

Notebooks:
Exploration.ipynb - Contient l'exploration du jeu de données et le clustering
Classification.ipynb - Contient les différents modèles de classification testés et les résultats

Python
classify.py - Contient le programme de classification final
params.pickle - Contient les paramètres nécessaires pour le programme

Texte:
Rapport.pdf - Rapport du projet
